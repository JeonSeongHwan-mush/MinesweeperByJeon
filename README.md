## MinesweeperByJeon
> 개발의 재미는 생각을 현실로 만드는 것에 있다고 생각한다. 

> 그러다 문득 지뢰찾기는 구조가 어떻게 되는걸까.. 생각하며 만들어본 나만의 지뢰찾기 게임이다.

> 완전한 게임이라기 보다는 10 x 10의 배열에서 10개의 지뢰를 랜덤으로 생성하는 코드까지 작성해보았다.

> 부족하지만 생각을 현실로 만들어 보았다는 것에서 성취감을 느껴보며 github에 기록해본다ㅎㅎ



#### 동작 화면 캡쳐
